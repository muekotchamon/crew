# Crew

Subcontractor compensation report — Next.js app with crew management, documents, and PDF preview/signing.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Stack

- [Next.js](https://nextjs.org)
- React PDF (`@react-pdf/renderer`)
